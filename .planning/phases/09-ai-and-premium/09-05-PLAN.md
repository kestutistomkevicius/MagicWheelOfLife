State: `messages: AiChatMessage[]`, `streaming: boolean`, `proposal: ScoreProposal | null`, `error: string | null`, `lastUserText: string` (for retry).

`loadHistory(categoryId)`: query `supabase.from('ai_chat_messages').select('role, content').eq('category_id', categoryId).eq('user_id', userId).order('created_at', { ascending: true })`. Map rows to `AiChatMessage[]` and `setMessages`.

`sendMessage(userText)`:
1. Append `{ role: 'user', content: userText }` to messages state.
2. Set `lastUserText = userText`, `streaming = true`, `error = null`.
3. Persist user message: `supabase.from('ai_chat_messages').insert({ user_id, category_id, role: 'user', content: userText })`.
4. Append `{ role: 'assistant', content: '' }` placeholder.
5. Get session token via `supabase.auth.getSession()`.
6. `fetch(${VITE_SUPABASE_URL}/functions/v1/ai-coach, { ... body: JSON.stringify({ categoryName, asisScore, tobeScore, messages: [...messages, userMsg] }) })`.
7. If `!res.ok`: set `error = 'AI response failed'`, `streaming = false`, return.
8. Stream body via `res.body.getReader()` + `TextDecoder`. Accumulate into `rawAccumulated`. On each chunk: update last assistant message content with `stripSentinel(rawAccumulated)`.
9. After stream ends: run `detectAndSetProposal(rawAccumulated)`. Persist final assistant message (stripped content) to `ai_chat_messages`.
10. Set `streaming = false`.

`detectAndSetProposal(text)`: regex match `\{"type":"score_proposal","asis":(\d+),"tobe":(\d+)\}`. If found: `setProposal({ asis, tobe })`.

`stripSentinel(text)`: remove the sentinel JSON pattern from the text (replace with empty string, trim).

`retry()`: calls `sendMessage(lastUserText)` if `lastUserText` is not empty.
