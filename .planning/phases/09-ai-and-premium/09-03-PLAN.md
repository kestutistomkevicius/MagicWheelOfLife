  <action>
Create the directory `supabase/functions/ai-coach/` if it does not exist, then create `index.ts`.

The function must:

1. Handle CORS preflight (OPTIONS) — return 200 with headers `Access-Control-Allow-Origin: *` and `Access-Control-Allow-Headers: authorization, content-type`

2. Validate the JWT from the `Authorization: Bearer <token>` header using `supabase.auth.getUser(token)`. Return 401 if missing or invalid.

3. Check the user's tier via `profiles` table. Return 403 if tier is not `'premium'`.

4. Parse the POST body: `{ categoryName, asisScore, tobeScore, messages }` where messages is the full conversation history as `{role, content}[]`.

5. Build the system prompt (the `buildSystemPrompt` function):
```
You are a thoughtful life coach helping a user reflect on their "${categoryName}" life area.
Their current self-assessment: As-Is score ${asisScore}/10, To-Be (goal) score ${tobeScore}/10.

Ask 2-4 reflective questions to understand their situation better before suggesting scores.
Keep responses concise (2-3 sentences per turn). Be warm and encouraging.

When you have enough context (after at least one user response), output your score suggestion in this EXACT format on its own line at the end of your message:
{"type":"score_proposal","asis":X,"tobe":Y}
where X and Y are integers 1-10 based on what they have shared.
Never suggest a score before asking at least one question.
```

6. Call Anthropic streaming:
```typescript
const stream = await anthropic.messages.create({
  model,
  max_tokens: 1024,
  stream: true,
  system: buildSystemPrompt(categoryName, asisScore, tobeScore),
  messages,
})
```

7. Pipe tokens to the browser via ReadableStream — iterate over `stream` events, emit `content_block_delta` / `text_delta` chunks.

8. Return `new Response(readable, { headers: { 'Content-Type': 'text/plain; charset=utf-8', 'Access-Control-Allow-Origin': '*', 'Transfer-Encoding': 'chunked' } })`

Set up local secrets before testing:
```bash
supabase secrets set ANTHROPIC_API_KEY=sk-ant-YOUR_KEY_HERE
supabase secrets set ANTHROPIC_MODEL=claude-haiku-4-5-20251001
```
